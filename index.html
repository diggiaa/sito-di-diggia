Project Description

This project focuses on remote monitoring of humidity and temperature, aligned with the theme “Health and Wellbeing at School”. Optimal classroom conditions play a key role in comfort and learning. Our IoT device allows real-time monitoring of environmental data.

Components Used

  • Arduino Uno
  • DHT11 Sensor
  • SCT-013-030 Sensor
  • PCB
  • 16x2 LCD Display
  • Resistors, jumper wires and breadboard
  • USB cable and power supply

How It Works

The process is simple: we connect and calibrate the sensor using a smart plug with different loads (phone, PC, charger, etc.). The system is then installed in all school electrical panels, enabling IoT-based monitoring of energy consumption. Data storage is managed through Firebase.
